CORE ENDPOINTS

GET  /api/trends          — Scored & ranked fashion trends
GET  /api/forecast        — 7/14/30-day ML projections
GET  /api/news            — Latest fashion news (20+ sources)
GET  /api/reddit          — Top Reddit fashion posts
GET  /api/google-trends   — Google Trends data
GET  /api/aesthetics      — Aesthetic movement scores
GET  /api/colors          — Color trend palette
GET  /api/brands          — Brand mention leaderboard
GET  /api/calendar        — Fashion event calendar
GET  /api/dashboard       — Full dashboard summary

AI ENDPOINTS (key required)

GET  /api/ai/trend/<name>   — AI analysis for one trend
GET  /api/ai/overview       — AI overview of top trends
GET  /api/ai/season         — AI seasonal outlook
GET  /api/search?q=<query>  — RAG search + AI analysis
POST /api/db/ingest         — Trigger full data ingest

KEY MANAGEMENT

POST /api/keys/register   — Register & get a new key
GET  /api/keys/verify     — Verify key & check quota
